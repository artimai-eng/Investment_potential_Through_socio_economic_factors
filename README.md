# Investment Decision Support System

A data-driven dashboard that evaluates country-level investment potential using socioeconomic indicators and recommends high-potential sectors for investment.

## Overview

This project leverages WHO socioeconomic data to assess the investment readiness of different countries. By analyzing indicators such as GDP, life expectancy, education levels, mortality rates, and vaccination coverage, the system generates investment recommendations and identifies promising sectors for business expansion.

## Objective

To support data-driven investment decisions by transforming complex socioeconomic data into actionable insights.

## Tech Stack

- Python
- SQL
- MySQL
- Streamlit


## Features

- Country-wise investment analysis
- Investment readiness scoring system
- Risk classification:
  - Invest
  - Moderate Opportunity
  - High Risk
- Sector recommendation engine
- Interactive dashboard interface
- MySQL database integration

## Key Indicators Used

- GDP
- Life Expectancy
- Schooling
- Adult Mortality
- Polio Vaccination Coverage

## Recommended Sectors

Based on country performance, the system recommends sectors such as:

- Healthcare & Insurance
- Education & Human Capital
- IT & Business Services
- Consumer & Retail
- Manufacturing & Industrial Production
- Public Health Infrastructure

## Impact

- Automated country-level investment assessment using socioeconomic indicators.
- Converted WHO data into actionable investment insights.
- Generated sector-specific recommendations to support strategic decision-making.
- Simplified complex data analysis through an interactive dashboard.
